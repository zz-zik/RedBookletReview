# 小红书评论数据集

## 介绍
我们利用爬虫来对小红书网页笔记进行爬取评论，然后讲爬取的数据保存到以当前时间命名的csv文件中，再经过数据清洗、预处理、标注等操作，最后分为了训练集、验证集、测试集比例为6:2:2。

## 文件结构如下：
RedBookletReview
├── RedBookletReview
  ├── dev.tsv
  ├── test.tsv
  ├── train.tsv
└── results
  └── RedBookletReview.tsv

## 说明
本数据集仅供学习交流，并非是专业人士标注，因此数据集的质量并不是特别高，如果你在训练的时候遇到效果不是很好的情况是正常的，可通过调整句子长度来提升。
